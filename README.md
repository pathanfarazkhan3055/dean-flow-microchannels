# Numerical Study of Dean Flow Formation in Curved Microchannels

## Overview
Developed and simulated 3D U-bent and spiral microchannel geometries in ANSYS Fluent to investigate Dean vortex formation and secondary flow behavior.

## Tools Used
- ANSYS Fluent
- CFD

## Key Parameters
- Reynolds Number: 50–100
- Dean Number: 6.11–26.8
- Velocity Range: 0.38–0.75 m/s

## Key Findings
- Investigated Dean vortex formation
- Analyzed secondary flow development
- Evaluated mixing enhancement in curved microchannels
- Studied fluid transport characteristics
